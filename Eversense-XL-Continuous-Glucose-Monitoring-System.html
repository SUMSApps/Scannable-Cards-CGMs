Calibration, data storage and result sharing

Calibration
During initialisation phase (immediately after implantation), 4 fingerstick tests are required. After this, daily calibration is required with 1 test twice per day

Stored data
Data can be synced from the mobile app to the Eversense Data Management System

Data is also stored in the mobile app and on the smart transmitter

Result sharing
Results can be shared with selected friends/family via the Eversense NOW app (iOS)

The Eversense Data Management System can be used by HCPs & users to analyse data
